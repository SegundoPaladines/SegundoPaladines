<h2 align="left">
    💼 What I Do
</h2>

**Zero Platform Engineer (ZPE)** · **XOFTIX S.A.S** · _Jan 2024 – Present_

I build production platforms where applied AI meets solid software architecture — from requirements analysis to scalable, containerized deployments.

**🤖 AI Engineering**

- Develop and orchestrate **multi-agent systems** to automate complex business processes.
- Build and optimize **RAG pipelines** to ground LLMs on corporate data and improve answer precision.
- Ship **computer vision** solutions for data extraction, pattern recognition and visual inspection.
- Expose AI capabilities through **Python + FastAPI** backends, integrated via **REST and async messaging**.
- Store and search **embeddings** on vector databases (**Qdrant · Pinecone · Weaviate · Milvus**).
- Apply **prompt engineering** to maximize response quality while cutting operational cost.
- Package and deploy with **Docker** on virtualized / private-cloud environments.

**🏗️ Software Architecture**

- Analyze functional and non-functional requirements and estimate size and complexity.
- Design the full system structure, its interfaces and evaluation criteria across the project lifecycle.
- Run global impact analysis and lead architecture **refactoring** when the design calls for it.
- Drive **performance analysis** and guarantee scalable, efficient and maintainable systems.
- Act as the technical bridge across developers, QA, requirements engineers and PMs, coordinating dev teams.
